# v2 Web Tools — Implementation Plan

## Overview

Implement ADR-0010 (web tools for experts) and ADR-0011 (nonce every structural fence + tighten forgery regex) on top of the merged v2 debate engine. Net delta: experts can `WebSearch` and `WebFetch` during R1 and R2; ballots are hardcoded tools-off; every structural fence in the prompt protocol carries the session nonce; forgery regex rejects exactly the nonce-bearing fence shape.

**Source of truth:** `docs/design/v2.md` (§4 D11 as amended, §4 D17–D21, §8 F13–F17), `docs/adr/0010-web-tools-for-experts.md`, `docs/adr/0011-amend-0008-nonce-every-fence.md`, `docs/design/v2-web-tools.md`.

## Context (from discovery)

- **v2 code base on main:** `pkg/debate/`, `pkg/prompt/`, `pkg/executor/`, `pkg/config/` — all settled after PR #4.
- **Files touched by this plan:**
  - `pkg/executor/executor.go` — `Request` gains `AllowedTools []string` and `PermissionMode string`.
  - `pkg/executor/claudecode/claudecode.go` — conditionally append `--allowedTools` and `--permission-mode` when non-empty.
  - `pkg/executor/mock/mock.go` — record the flag set so F13/F14/F17 can assert on it.
  - `pkg/config/config.go` — `Profile` gains `Tools *ToolsBlock`; add `ToolsBlock{WebSearch, WebFetch bool}`.
  - `pkg/config/loader.go` — accept the `tools:` block in yaml, snapshot it in `profile.snapshot.yaml`.
  - `pkg/prompt/injection.go` — tighten `delimiterLineRE` to `(?m)^=== .*\[nonce-[0-9a-f]{16}\] ===[ \t\r]*$`; update doc comments. Update `ScanQuestionForInjection` to use the same regex (benign markdown in the operator's question now passes).
  - `pkg/prompt/expert.go` — `BuildExpert(roleBody, question, nonce string)` — new mandatory nonce parameter; USER QUESTION fences carry `[nonce-<hex>]`.
  - `pkg/debate/vote.go` — `buildBallotPrompt(ballotBody, question, aggregateMD, nonce string)` — USER QUESTION + CANDIDATES fences carry `[nonce-<hex>]`; `RunBallot` spawns ballots with `AllowedTools: nil`.

  - Existing tests updated: `pkg/prompt/injection_test.go`, `pkg/prompt/expert_test.go`, `pkg/debate/rounds_test.go`, `pkg/debate/vote_test.go`, `pkg/config/embed_test.go`, `pkg/config/loader_test.go`.
- **Prompts updated in place (content only, no filename change):**
  - `defaults/prompts/independent.md` — add research discipline + citation rules.
  - `defaults/prompts/peer-aware.md` — add verification discipline + explicit sycophancy resistance.

**Key contracts to preserve:**

- v1 CLI (`council "q"`, `-p`, `--profile`, `-`, `-v`) still works.
- Executor interface doc contract unchanged (fields are additive).
- v1 `default.yaml` (no `tools:` block, 180s timeouts) still loads against the v2+tools binary and produces v1-equivalent behavior.
- Atomic verdict.json write unchanged; no schema version bump.
- Folder-as-database (ADR-0003) unchanged.
